#Termchat
Многопользовательский чат в вашем терминале

## Запуск сервера
Выполним команду, которая создаст исполняемый файл для запуска сервера:
```
$ make server
```
Запускаем сервер:
```
$ ./server
```

## Запуск клиента
Выполним команду, которая создаст исполняемый файл для запуска клиента:
```
$ make client
```
Запускаем клиент:
```
$ ./client
```
